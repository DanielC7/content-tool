Content Tool

Instructions:
1) Upload an Excel file according to the format:
- Excel file must be of type .xlsx.
- Excel file must contain the 3 columns - "Id", "Translated", "Language", in that order.
- "Id" column should contain only numbers, "Translated" column may be empty or filled, and "Language" column should contain only strings.
2) Open the language dropdown list, choose the language you want to insert its translated data into the DB, and click 'Submit'.
3) Copy the SQL script using the designated 'Copy' button.
